Bouncing Text Screensaver

A simple screensaver-style program built with Python and Pygame.
It displays bouncing text that moves around the screen and reverses direction when it hits the edges.

---

Features

Smooth bouncing text animation

Adjustable screen size and speed

Runs at a stable 60 FPS

Minimal and beginner-friendly Pygame example

---

Requirements

Python 3.x

Pygame

Install Pygame with:

pip install pygame

---

How to Run

python main.py
